# Google Workspace Setup

A skill for setting up and configuring Google Workspace tools for clients.

## What this covers
- Gmail setup and configuration
- Google Calendar setup
- Google Drive setup
- OAuth credentials configuration
- MCP server integration with Google Workspace

## Files
- `SKILL.md` — main skill instructions
- `scripts/client_secret.json` — OAuth credentials template

## Usage
Use this skill when a client needs Google Workspace tools connected and configured.
